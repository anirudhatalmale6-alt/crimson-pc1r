WHAT'S ALREADY DONE IN THE .cd32 FILE:
- Hardware: CR1000-10000 (800x480 display)
- Ethernet IP: 192.168.16.30 / Netmask: 255.255.255.0
- Communications: AB L5k Plus driver
  PLC1 at 192.168.16.20:44818 (Ethernet/IP)
- 181 Data Tags imported from PC1R.L5K
- 3 Display Pages: Main, IO_Summary, Alarm
- Tags you need are under: PLC1 > Controller_Tags

============================================================
PART 1: IO_SUMMARY DISPLAY PAGE
============================================================

RECOMMENDED LAYOUT (800 x 480 screen):
- Title bar at top: "I/O SUMMARY" (centered, large font)
- 4 columns across the page, one per I/O module
- Each I/O point = small indicator (color changes ON/OFF)
  with label text beside it

STEP-BY-STEP:

1. Double-click "IO_Summary" page in Navigation Pane
2. Double-click the black canvas to enter Edit mode
   (Menu bar should show: Mode, Behavior, Text, Data,
    Transform, Arrange, Organize, Link)

--- ADD TITLE ---
3. In Resource Pane (right side), click "Primitives" tab
4. Click "Core Primitives" category to expand it
5. Find "Text Box" and drag it onto the canvas
6. Position at top center: roughly (200, 2) to (600, 28)
7. Press F2 or use Text menu to edit the text
8. Type: I/O SUMMARY
9. Set font to something like Hei 14 Bold, White text
10. Click OK

--- ADD I/O INDICATORS ---
For each I/O point, you need an INDICATOR from
the "Indicators" category in the Primitives panel.

The fastest method:
a) Drag ONE "Indicator" from Primitives > Indicators onto canvas
b) Right-click it > Properties
c) On the "Figure" tab, set ON color = Green, OFF color = Red
d) On the "Text" tab, type the I/O description
e) On the "More" tab, set position and size
   (recommended: Width=190, Height=16 for each indicator)
f) Link it to a data tag:
   - Click the element, then use Data menu > Add (Ctrl+F2)
   - Or on the properties Figure tab, set the "Tag" field
   - Browse to PLC1 > Controller_Tags > [module] > DATA > [bit]
g) Click OK
h) Copy this indicator (Ctrl+C), paste (Ctrl+V)
i) Edit the copy's text and tag link for the next I/O point

============================================================
PART 2: ALARM DISPLAY PAGE
============================================================

RECOMMENDED LAYOUT:
- Title at top: "ALARM FAULTS"
- Single column list, or 2 columns
- Each alarm = indicator that turns RED when fault is active
- Use "Indicators" from Primitives panel

--- STEP-BY-STEP ---
1. Double-click "Alarm" page in Navigation Pane
2. Double-click canvas to enter Edit mode
3. Add title text "ALARM FAULTS" at top
4. For each alarm, drag an Indicator from Primitives
5. Set ON color = RED, OFF color = dark gray or green
6. Link to the N7 bit tag as shown below

============================================================
QUICK INDICATOR SETUP (FASTEST METHOD)
============================================================

1. Open Primitives tab in Resource Pane (right panel)
2. Click "Indicators" category
3. Drag any indicator style onto the canvas
4. Right-click the indicator > Properties
5. TEXT tab:
   - Type the description (e.g., "VICES CLOSE SOL")
   - Set font size small enough to fit (Hei 8 or 10)
   - Text Color: White
6. FIGURE tab:
   - This is where you set the ON/OFF colors
   - For I/O: ON=Green, OFF=Dark Gray
   - For Alarms: ON=Red, OFF=Dark Gray
   - TAG field: browse to the correct PLC1 tag path
7. MORE tab:
   - Set exact position (X, Y) and size (Width, Height)
   - Recommended size: W=190, H=16 per indicator
8. Click OK

COPY SHORTCUT:
- After setting up one indicator perfectly, Ctrl+C to copy
- Ctrl+V to paste a copy
- Move it below the first one
- Right-click > Properties to change just the text and tag
- This is MUCH faster than creating each from scratch
